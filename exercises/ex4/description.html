1-bit adder with carry input (full adder)

In this exercise, we will create a 1-bit adder with a carry input and a carry output (also named full adder). You will need to create an entity that:

  • uses a, b, cin as its input
  • outputs its sum to s and carry to cout
You can use the signals carry1, carry2 and s1 below to get your final ouputs.
